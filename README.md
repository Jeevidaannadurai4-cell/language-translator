# 🌍 Language Translator Web Application

## 🚀 Live Demo

🔗 https://jeevidaannadurai4-cell.github.io/language-translator/

## 📌 Project Overview

Language Translator is a web-based application that helps users translate text from one language to another using a translation API.

The project is developed using HTML, CSS, and JavaScript with a clean and responsive user interface.

## ✨ Features

✅ Language Translation  
✅ English ↔ Tamil Translation  
✅ Copy Translation Text  
✅ Clear Input and Output  
✅ Character Counter  
✅ Dark Mode Support 🌙  
✅ Translation History 📜  
✅ Responsive Design for Mobile and Desktop  

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Translation API
- Browser Storage (LocalStorage)

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in a browser
3. Enter text to translate
4. Select source and target languages
5. Click the Translate button
